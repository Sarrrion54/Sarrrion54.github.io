Online учебники и/или справочники по HTML, CSS, Javascript, PHP.

Краткая информация:

Htmlbook.ru предназначен для изученния HTML и CSS. Здесь
можно познакомиться с большинством тегов HTML и
прочитать справочник CSS. Всё объясняется понятным языком.
Минусом является большое количевство рекламы.

Краткая информация:

Htmlacademy-здесь можно познакомиться с HTML, CSS, JavaScript.
Здесь много познавательных курсов (как платных, так и бесплатных)
Рассказывается всё понятным языком.

Краткая информация:

MDN web docs-здесь помимо CSS можно познакомиться с HTML и JS.
На этом сайте можно лишь узнать основы CSS. Прост в понимании.

Краткая информация:

На Indigo есть "Современный учебник CSS" — это онлайн-книга, призванная
ознакомить веб-разработчиков с современными способами создания сайтов,
в том числе и адаптивных. В ней рассматриваются новейшие методы
применения CSS и HTML. Материал рассчитан на тех, кто уже знаком с
HTML хотя бы на начальном уровне.
